# Обзор версий

Здесь представлена сводная информация о недавних выпусках и обновлениях **Comindware Platform**, включая краткие аннотации к каждой версии, чтобы помочь вам более эффективно использовать все возможности продукта.

- **[Версия 6.0](https://kb.comindware.ru/article.php?id=5741)**: представлен интеллектуальный ассистент (ИИ) для работы с системой на естественном языке, умной навигации, экспресс-анализа данных, автоматического заполнения форм и выполнения бизнес-операций через low-code-агентов в чате; добавлены подсказки для настройки ИИ и поддержка форматирования Markdown в чатах с ИИ.
