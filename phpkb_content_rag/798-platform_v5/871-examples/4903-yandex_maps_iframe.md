# Яндекс Карты. Настройка отображения в iframe

Здесь приведён пример настройки отображения заранее созданной Яндекс Карты в поле на форме **Comindware Platform**.

Такой вариант можно использовать, например, чтобы показать на карточке организации карту с офисами, складами или другими объектами.

## Настройка карты

1. Создайте карту с необходимым количеством меток на сайте [конструктора Яндекс Карт](https://yandex.ru/map-constructor/).
2. Нажмите кнопку «**Сохранить и продолжить**».
3. Выберите размеры карты для отображения в `iframe`.
4. Нажмите кнопку «**Получить код карты**».
5. Выберите вариант `iframe` и скопируйте код.

## Настройка атрибута

1. В нужном шаблоне записи создайте атрибут типа «**Текст**».
2. Для атрибута выберите формат отображения «**HTML-текст**».
3. Установите флажок «**Вычислять автоматически**».
4. В поле «**Вычисляемое значение**» выберите «**Формула**».
5. Вставьте скопированный код `iframe` в двойные кавычки.
6. Внутри кода `iframe` замените все двойные кавычки на одинарные.

Совет

Если оставить двойные кавычки внутри кода `iframe`, формула будет обработана некорректно, так как они завершат строковый литерал раньше времени.

## Проверка результата

1. Вынесите атрибут с HTML-кодом карты на форму.
2. Откройте запись.
3. Убедитесь, что карта отображается на форме.
