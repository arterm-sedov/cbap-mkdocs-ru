# Шаблон организационной единицы

## Определения

- **Шаблон организационной единицы** — это специальный [шаблон записи](https://kb.comindware.ru/article.php?id=4759), в котором хранятся особые атрибуты оргединиц.
- Шаблоны организационных единиц используются для настройки и хранения записей, описывающих организационную структуру предприятия.
- Например, шаблон оргединицы *«Департаменты»* может содержать записи: *«Департамент хозяйственного обеспечения», «Организационно-правовой департамент».*

Примечание

В каждом [приложении](https://kb.comindware.ru/article.php?id=4714) предусмотрен один *системный шаблон организационной единицы*. Он создается автоматически при создании приложения.

Удалить или создать новый шаблон оргединицы нельзя.

## Свойства шаблона оргединицы

У шаблона организационной единицы предусмотрены только [общие свойства](https://kb.comindware.ru/article.php?id=4756).
