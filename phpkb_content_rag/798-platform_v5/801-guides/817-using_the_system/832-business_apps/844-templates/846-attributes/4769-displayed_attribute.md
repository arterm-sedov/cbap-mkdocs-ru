# Атрибут-заголовок записей шаблона

## Определения

В шаблоне можно выбрать атрибут, значение которого будет отображаться в качестве заголовка для записей:

- на формах и в таблицах — в полях атрибутов типа «**Запись**», «**Аккаунт**», «**Роль**», «**Организационная единица**», связанных с данным шаблоном.
- в заголовках форм данного шаблона.

В качестве заголовка записей шаблона можно использовать атрибут любого следующих типов:

- [Аккаунт](https://kb.comindware.ru/article.php?id=4774)
- [Дата и время](https://kb.comindware.ru/article.php?id=4777)
- [Длительность](https://kb.comindware.ru/article.php?id=4775)
- [Текст](https://kb.comindware.ru/article.php?id=4768)
- [Число](https://kb.comindware.ru/article.php?id=4773)

По умолчанию в качестве заголовка записей используются следующие системные атрибуты:

- в [шаблоне аккаунта](https://kb.comindware.ru/article.php?id=4757) — «**Ф. И. О.**» (`fullName`);
- в [шаблоне записи](https://kb.comindware.ru/article.php?id=4759) — «**ID**» (`id`);
- в [шаблоне оргединиц](https://kb.comindware.ru/article.php?id=4755) — «**Название**» (`unitName`);
- в [шаблоне роли](https://kb.comindware.ru/article.php?id=4760) — «**Название**» (`roleName`).

_![Пример отображения имени заявителя в заголовке формы вместо ID записи](/platform/v5.0/business_apps/templates/attributes/img/displayed_attribute_example.png)_

## Выбор атрибута в качестве заголовка записей шаблона

Установите флажок «**Использовать как заголовок записей**» в [свойствах атрибута](https://kb.comindware.ru/article.php?id=4772#attributes_configure).

Примечание

Для шаблона можно указать только один атрибут в качестве заголовка записей, поэтому после установки флажка «**Использовать как заголовок записей**», он будет автоматически снят у атрибута, который ранее был выбран в качестве заголовка записей.

_![Флажок «Использовать как заголовок записей»](/platform/v5.0/business_apps/templates/attributes/img/displayed_attribute.png)_
