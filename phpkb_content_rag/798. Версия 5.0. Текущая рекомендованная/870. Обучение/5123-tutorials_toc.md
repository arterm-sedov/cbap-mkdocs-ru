# Обучение. Содержание раздела

Здесь представлены уроки для быстрого освоения **Comindware Platform**:

- *[Курс «Аналитик. Ознакомительный»](https://kb.comindware.ru/article.php?id=5112)* — создайте своё первое приложение с помощью **Comindware Platform** на примере управления персоналом.
- *[Курс «Корпоративная архитектура. Ознакомительный»](https://kb.comindware.ru/article.php?id=5124)* — освойте основы моделирования корпоративной архитектуры с помощью **Comindware Platform Enterprise**.
- *[Курс «Управление автопарком»](https://kb.comindware.ru/article.php?id=5085)* — создайте своё первое приложение с помощью **Comindware Platform** на примере управления корпоративным автопарком. Примечание: вместо этого курса рекомендуется проходить курс *«[Аналитик. Ознакомительный](https://kb.comindware.ru/article.php?id=5112)»*.
