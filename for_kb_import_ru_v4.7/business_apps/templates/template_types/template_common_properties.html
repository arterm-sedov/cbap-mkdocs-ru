Перечень общих свойств шаблонов

У шаблонов всех типов предусмотрены перечисленные ниже основные свойства.

  • ID — уникальный идентификатор шаблона, формируется автоматически.
  • Название — наименование шаблона. Обязательное поле.
  • Системное имя — уникальное имя шаблона. Обязательное поле. Не должно начинаться с цифры. Рекомендуется использовать буквы латинского алфавита, цифры и символ «_». Заполняется автоматически по названию.
  • Описание — необязательный комментарий относительно назначения шаблона.
  • Приложение — приложение, в котором используется шаблон.
  • Конфигурация обсуждений — настройка обсуждений на формах.
