Введение

В Comindware Platform предусмотрена возможность двухфакторной аутентификации пользователей мобильного приложения Comindware Mobile.

Для использования двухфакторной аутентификации необходимо включить эту функцию в глобальной конфигурации, настроить подключение для отправки эл. писем с кодом подтверждения и включить двухфакторную аутентификацию для аккаунтов, которым она необходима.

Внимание!

Здесь представлены инструкции по настройке двухфакторной аутентификации для мобильного приложения Comindware Mobile. То есть настройки не повлияют на вход в веб-версию Comindware Platform.

Двухфакторную аутентификацию в веб-версии Comindware Platform можно настроить средствами Keycloak. См. «Аутентификация через Keycloak и OpenID Connect. Настройка подключения и служб».

Процесс двухфакторной аутентификации

  1. При попытке входа на сервер Comindware Platform в мобильном приложении Comindware Mobile пользователю будет отправлено письмо с кодом подтверждения.
  2. Пользователь должен ввести полученный код из четырёх цифр в течение 5 минут после его отправки. В отношении кода подтверждения действуют следующие ограничения:
    • повторно отправить код можно с интервалом в 1 минуту;
    • неверный код можно ввести не более трёх раз.

Порядок настройки

  1. Откройте страницу «Администрирование» – «Глобальная конфигурация».
  2. В разделе «Двухфакторная аутентификация» выберите пункт «Включено».
  3. Перезапустите экземпляр ПО Comindware Platform.

Для этого обратитесь к системному администратору.

  1. Настройте подключение типа «Отправка эл. почты из процесса» и установите в его свойствах флажок «Использовать по умолчанию».
  2. Откройте страницу «Администрирование» – «Аккаунты».
  3. В списке аккаунтов установите флажки выбора для аккаунтов, которым требуется включить двухфакторную аутентификации.
  4. Нажмите кнопку «Двухфакторная аутентификация».
  5. Во всплывающем окне «Настройка двухфакторной аутентификации» выберите пункт «Через эл. почту» и нажмите кнопку «Закрыть».
  6. После включения двухфакторной аутентификации для новых аккаунтов будет автоматически устанавливаться режим двухфакторной аутентификации «Через эл. почту».
