Здесь приведён пример импорта связанных данных из файла Excel в Comindware Platform.

Такой импорт удобно использовать, когда требуется загрузить записи в связанный шаблон и сразу отобразить их в таблице на форме основной записи.

Принцип работы

Таблица на форме записи отображает записи из связанного шаблона. Каждая строка такой таблицы является отдельной записью, которая ссылается на основную запись.

Чтобы заполнить таблицу на форме с помощью Excel:

  1. Импортируйте файл Excel в связанный шаблон записи.

  2. Укажите в отдельном столбце файла значение для связи с основной записью:

    • ID основной записи;
    • значение её атрибута-заголовка, если оно используется как формат сопоставления Name.

После импорта записи будут связаны с основной записью, а таблица на её форме заполнится автоматически.

Пример настройки

В примере требуется импортировать заказы клиентов из Excel.

В приложении созданы два шаблона записи:

  • Клиенты — основной шаблон записи;
  • Заказы — связанный шаблон записи.

В шаблоне «Заказы» настроен атрибут типа «Запись», который ссылается на шаблон «Клиенты». В шаблоне «Клиенты» настроен связанный атрибут «Заказы клиента» с включённым хранением нескольких значений.

Подготовка файла Excel

Подготовьте файл Excel со столбцами заказов и отдельным столбцом для связи с клиентом.

В столбце «Клиент» укажите названия клиентов, уже созданных в Comindware Platform. Эти значения должны совпадать со значениями атрибута-заголовка записей в шаблоне «Клиенты».

Импорт заказов

  1. Откройте шаблон записи «Заказы».
  2. Перейдите на вкладку «Импорт».
  3. Загрузите подготовленный файл Excel.
  4. Настройте сопоставление столбцов Excel с атрибутами шаблона «Заказы».
  5. Для столбца «Клиент» выберите атрибут типа «Запись», который ссылается на шаблон «Клиенты».
  6. В поле «Формат» для столбца «Клиент» выберите Name.
  7. Для столбца «Сумма» выберите формат «Валюта».
  8. Нажмите кнопку «Загрузить».

Результат

После импорта в шаблоне «Заказы» появятся новые записи со связью с соответствующими клиентами.

Таблица «Заказы клиента» на форме записи в шаблоне «Клиенты» заполнится автоматически.
