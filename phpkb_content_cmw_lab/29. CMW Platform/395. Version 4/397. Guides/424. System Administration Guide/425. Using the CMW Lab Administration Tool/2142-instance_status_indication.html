CMW Platform Instance Status Indication

1. Launch the Administration Tool (see Launching the CMW Lab Administration Tool).

2. The first column of the Product instances list can show status icons:

  • A warning triangle  — the instance configuration was not found. Try changing the system instance file paths (see Configuring a CMW Platform Instance {Article-ID:2141}).
  • A stop sign  — the website or IIS application pool for the instance is stopped. To start the website, right-click the instance and select Start website from the context menu.
