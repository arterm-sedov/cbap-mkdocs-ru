Available Operations

The button list provides the following operations and buttons.

  • Select — check a box in the first column to select the button.
  • Open — double-click a button row to open the button designer.
  • Create — create a new button.
  • Delete — delete a button (displayed when one button is selected).

  • Search — click this button and enter keywords in the Search field. The lines containing the keywords will be shown.
