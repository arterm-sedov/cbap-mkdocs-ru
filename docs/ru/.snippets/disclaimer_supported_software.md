## Поддерживаемые версии ПО {: .admonition-title #supported_software}

- Компания **{{ companyName }}** осуществляет поддержку двух версий ПО:
    - текущей рекомендованной версии для установки и использования — это версия {{ productVersion }};
    - предыдущей поддерживаемой версии — это версия {{ previousSupportedVersion }}.
- Поддержка версии {{ previousSupportedVersion }} будет осуществляться до момента выхода новой текущей рекомендованной версии (например, {{ productVersionSupportEndExample }}), после чего поддержка версии {{ previousSupportedVersion }} будет прекращена.
- Версии ПО ниже {{ previousSupportedVersion }} не поддерживаются и не рекомендованы к установке, пользователи используют их на свой страх и риск.
- Для получения технической поддержки пользователям ПО версий ниже {{ previousSupportedVersion }} следует обновить ПО до текущей рекомендованной версии.
