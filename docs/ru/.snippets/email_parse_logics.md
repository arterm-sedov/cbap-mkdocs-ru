!!! tip "Извлечение данных из содержимого эл.&nbpsp;писем для определения типа запроса"

    В зависимости от сценария использования может потребоваться разбор темы или тела письма для определения типа запроса или варианта обработки (новый/существующий экземпляр процесса).

    Для разбора содержимого эл.&nbpsp;писем используйте **формулы** или язык **N3**:

    - **парсинг темы письма** — извлечение идентификаторов процессов, типов запросов, ключевых слов;
    - **анализ текста письма** — поиск специфических данных, структурированной информации;
    - **определение отправителя** — анализ адреса эл.&nbsp;почты для маршрутизации запросов.
