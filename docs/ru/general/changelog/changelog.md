# Обзор версий {: changelog}

Здесь представлена сводная информация о недавних выпусках и обновлениях **{{ productName }} 5**, включая краткие аннотации к каждой версии, чтобы помочь вам более эффективно использовать все возможности продукта.

- **[Версия 5.0 / 31.12.2025][release_notes_5.0.20251231]**: добавлены параметры конфигурации для управления созданием топиков {{ apacheKafkaVariants }}; реализована поддержка сервисов на Service Host (API gateway); улучшена система обсуждений с системными уведомлениями; расширены возможности адаптеров (увеличение времени ожидания ответа, автоматический запуск); добавлен виджет выбора цвета для мобильного приложения; улучшены дашборды и рабочий стол; реализован сбор объектов через Object API с использованием нативных запросов; улучшена конфигурация кэшей; повышена стабильность и производительность; устранены известные ошибки.
- **[Версия 5.0 / 10.10.2025][release_notes_5.0.20251010]**: реализованы значительные улучшения в **{{ productNameArchitect }}**, дашбордах и поиске; расширены возможности администрирования (синхронизация с сервером каталогов, замещения); улучшены интеграции и пути передачи данных; повышены удобство и стабильность мобильного приложения.
- **[Версия 5.0 / 24.03.2025][release_notes_5.0.13334]**: улучшена работа диаграмм в модуле «{{ productNameArchitect }}»; усовершенствована публикация пользовательских адаптеров; оптимизирована работа мобильного приложения; улучшена инфраструктура для повышения его стабильности и производительности работы ПО; устранены известные ошибки.
- **[Версия 5.0 / 10.12.2024][release_notes_5.0]**: расширены возможности настройки **{{ productName }}**; усовершенствованы моделирование корпоративной архитектуры, интерфейс пользователя, интеграция с внешними системами и окружением, скрипты установки ПО, мониторинг, журналирование и аутентификация, фильтрация и глобальный поиск, чаты, управление версиями с помощью Git и Excel; повышена стабильность управления версиями и резервного копирования; реализованы поддержка чертежей в формате DWG, QR-кодов и NFC-меток, электронных подписей и электронного документооборота, интеграция с офисными приложениями.
