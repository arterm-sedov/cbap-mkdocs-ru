# Сведения о выпуске 5.0  от 09.06.2026 {: #release_notes_5.0.20260609 }

## Введение {: #release_notes_5.0.20260609_intro }

**{{ productName }}** версии 5.0  от 09.06.2026 включает ряд новых возможностей, улучшений и исправлений по сравнению с [версией 5.0 от 31.12.2025][release_notes_5.0.20251231].

В этом выпуске основное внимание уделено клиентским решениям, поиску, кластеру, новому адаптеру передачи данных через {{ apacheKafkaVariants }} и расширенным журналам.

Общий скоуп по изменениям и доработкам собран в коллекции «Включенные в релиз задачи и обращения»: https://support.comindware.ru/#form/oa.10/form.6633/10499104

!!! warning "Обновление до версии 5.0 "

    Из-за изменения структуры конфигурационных файлов обновление до версии 5.0  следует выполнять только посредством создания нового экземпляра ПО и подключения к нему базы данных обновляемого экземпляра.

## Ключевые изменения {: #release_notes_5.0.20260609_key_changes }

- **Клиентские решения:** выполнены приоритетные доработки и улучшения по задачам, выявленным при эксплуатации клиентских решений. <!-- ASAP -->
- **Поиск, таблицы и данные:** улучшены фильтрация, детализация данных на виджетах, перенос текста в таблицах, фиксация списков на формах и работа древовидных таблиц коллекций. <!-- 10245491, 10394348, 1880799, 10516611 -->
- **Кластер:** повышена стабильность работы API Gateway в кластерной конфигурации. <!-- 10405663 -->
- **Интеграции и {{ apacheKafkaVariants }}:** добавлены адаптеры импорта и экспорта данных через {{ apacheKafkaVariants }}, доработана архитектура адаптеров, улучшены SQL-коннектор и импорт данных из Excel. <!-- 10560845, 10366590, 10557913, 10498881 -->
- **Расширенные журналы:** добавлено журналирование действий над аккаунтами и группами в `IDM.log`, улучшено логирование сессий и лицензий. <!-- 10613670, 10594400, master_add_license_to_audit -->
- **Интерфейс и мобильное приложение:** улучшены таблицы, наборы данных, карточки, формы, быстрые фильтры, графики, виджеты, документы и сценарии работы с большими списками. <!-- 10391945, 10464084, 10490255, 10490522, 10534263 -->
- **{{ productNameArchitect }} и процессы:** повышена стабильность работы диаграмм, подпроцессов, вызываемых процессов, автоматического размещения элементов, версий приложений и выполнения экземпляров процессов. <!-- 10260209, 10294885, 10434086, 10454804, 10477491, 10506112, 10555583, 10572393, 10578802, 10578852, 10591839 -->
- **Администрирование и безопасность:** доработаны аутентификация, OpenID, LDAPS, SSO, HTML-санитизация, лицензирование, аудит, сведения о сессиях, синхронизация пользователей и разрешений ролей. <!-- 10323960, 10366756, 10484029, 10497077, 10507284, 10532973, 10583420, 10610192, 10340067 -->
- **Инфраструктура и производительность:** обновлён {{ apacheIgniteVariants }}, оптимизированы транзакции, предикаты, кэши, очереди, фоновые задачи, обновление версий, резервное копирование и работа с {{ apacheKafkaVariants }}. <!-- Ignite 2.18, 10349591, 10471489, 10478953, 10502726, 10554584, 10565778, 10590841, kafka threads leak, reducedMemoryUsageOnUpgrade -->

    - добавлено API для просмотра сессий и лицензий; <!-- master_license_api_for_keys -->
