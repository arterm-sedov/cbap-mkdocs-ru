# Сведения о выпуске 6.0 {: #release_notes_6.0 }

!!! warning "Черновик"
    Этот документ подготовлен по коммитам ветки `master` репозитория продукта за период с 01.01.2026 по 27.05.2026.  
    Состав и формулировки изменений будут уточняться перед публикацией выпуска.

## Введение {: #release_notes_6.0_intro }

**{{ productName }}** версии 6.0 включает ряд новых возможностей, улучшений и исправлений по сравнению с [версией 5.0 от 31.12.2025][release_notes_5.0.20251231].

В этом выпуске мы сосредоточились на:

- внедрении искусственного интеллекта;
- создании low-code-агентов для обработки бизнес-запросов на естественном языке;
- развитии интерфейса;
- мобильном приложении;
- **{{ productNameArchitect }}**;
- интеграциях;
- администрировании;
- производительности и устойчивости инфраструктуры.

Значительная часть изменений направлена на:

- взаимодействие с ИИ;
- выполнение бизнес-операций из чата с ИИ под контролем сценариев;
- более предсказуемую работу с большими объёмами данных;
- стабильное выполнение процессов;
- улучшение пользовательских сценариев на формах, страницах и в мобильном приложении;
- развитие средств настройки, журналирования и диагностики.

## Новые возможности и улучшения {: #release_notes_6.0_new_and_improved }

### Интеллектуальный ассистент (ИИ) {: #release_notes_6.0_ai }

В **{{ productName }}** версии **{{ productVersion }}** представлен встроенный интеллектуальный ассистент на базе искусственного интеллекта (ИИ), который существенно упрощает и ускоряет взаимодействие пользователей с системой за счёт распознавания запросов на естественном языке.

#### Интеграция и контекстное взаимодействие {: #release_notes_6.0_ai_integration }

- Ассистент обрабатывает текстовые запросы пользователей в режиме реального времени.
- Ассистент автоматически распознаёт контекст обращения (текущее приложение, роль и права доступа пользователя), обеспечивая точные и персонализированные ответы.

#### Распознавание намерений пользователя {: #release_notes_6.0_ai_intent }

Интеллектуальный алгоритм анализирует суть запроса и определяет необходимое действие:

- поиск и переход к требуемым разделам, отчётам или записям;
- автоматическое заполнение и редактирование записей и форм;
- предоставление справочных ответов на общие вопросы.

#### Умная навигация и экспресс-анализ данных {: #release_notes_6.0_ai_navigation }

Ассистент позволяет переходить по интерфейсу и управлять таблицами с помощью простых текстовых команд:

- открытие списков задач, карточек, дашбордов и виджетов;
- управление отображением данных — пользователь может попросить отсортировать, сгруппировать или отфильтровать текущую таблицу (например, _«покажи только активные договоры»_ или _«отсортируй по дате»_).

#### Автоматическое заполнение форм {: #release_notes_6.0_ai_forms }

- Реализовано создание и редактирование записей по текстовому описанию.
- Ассистент самостоятельно извлекает ключевую информацию из запроса пользователя и автоматически заполняет соответствующие поля формы, снижая нагрузку при рутинном вводе данных.

#### Low-code-агенты и операции в чате {: #release_notes_6.0_ai_low_code_agents }

- Реализована возможность строить ИИ-агентов на основе сценариев: сценарий получает сообщение из системного чата, передаёт в LLM бизнес-контекст и выполняет действия согласно настроенной логике.
- Ответы ИИ могут дополняться операциями в виде кнопок, чтобы пользователь мог сразу открыть запись, таблицу, задачу или дашборд и продолжить работу без ручного поиска.

#### Надежность и прозрачность работы {: #release_notes_6.0_ai_reliability }

- Все действия, выполняемые ИИ-ассистентом, проходят обязательную проверку перед применением.
- Запросы и результаты работы ИИ регистрируются в системных журналах, что обеспечивает безопасность, полный аудит и контролируемость автоматизированных операций.

### Подсказки для настройки ИИ {: #release_notes_6.0_ai_hints }

- Добавлены наглядные подсказки по настройке подключений и путей передачи данных для ИИ.

### Чаты {: #release_notes_6.0_chats }

- Реализована поддержка форматирования Markdown в чатах с ИИ.
