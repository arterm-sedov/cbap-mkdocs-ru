# Сведения о выпуске 5.0 от 31.12.2025 {: #release_notes_5.0.20251231 }

!!! danger "Внимание!"

    Если ваша версия **{{ productName }}** выпущена после 15.01.2026, её поведение может отличаться от версии, выпущенной 31.12.2025.

## Введение {: #release_notes_5.0.20251231_intro }

**{{ productName }}** версии 5.0 от 31.12.2025 включает ряд новых возможностей, улучшений и исправлений по сравнению с [версией 5.0 от 10.10.2025][release_notes_5.0.20251010].

## Ключевые изменения {: #release_notes_5.0.20251231_key_changes }

- **Интеграция с внешними системами:** оптимизировано взаимодействие со сторонними сервисами. <!-- 1613345 -->
- **Инициализация подключений:** оптимизировано подключение к внешним сервисам при запуске системы. <!-- 1936565 -->
- **Обсуждения:** добавлены уведомления о новых сообщениях, улучшены счётчики непрочитанных сообщений, улучшена внутренняя реализация обсуждений. <!-- git pull 14376 -->
- **Мобильное приложение:** улучшена работа ряда элементов интерфейса.
- **Рабочий стол и дашборды:** оптимизировано отображение графиков, усовершенствована настройка виджетов.
- **Мониторинг производительности:** функции мониторинга и просмотра показателей производительности перенесены на новый дашборд и в расширенные журналы (ранее находились на странице «Администрирование»).
- **Веб-интерфейс:** доработано отображение данных и фильтрация в таблицах.

## Новые возможности и улучшения {: #release_notes_5.0.20251231_new_and_improved }

### Интерфейс {: #release_notes_5.0.20251231_ui }

- **Рабочий стол, дашборды и страницы:**
    - оптимизирован контроль размеров графиков на рабочем столе; <!-- 10261293 -->
    - доработана прокрутка макета в конструкторе страниц; <!-- 10392976 -->
    - исправлено сохранение быстрых фильтров на страницах; <!-- 10411897 -->
    - обновлена проверка быстрых фильтров; <!-- 10365804 -->
    - усовершенствовано отображение атрибутов при настройке детализации данных диаграммы; <!-- 10410978 -->
    - оптимизирована цветовая схема, наглядно показывающая прохождение токенов по диаграмме экземпляра процесса;
    - устранено дублирование виджетов в колонках при редактировании страниц. <!-- 10402713 -->
**Таблицы:**
    - повышена производительность поиска записей при настройке фильтров для столбцов таблиц; <!-- 10433067 -->
    - реализован перенос слов в заголовках столбцов для улучшения отображения длинных названий; <!-- 10436143 -->
    - повышена производительность обработки списков с большим количеством строк. <!-- performance improvement -->
- **Диалоговые окна:** кнопки во всех диалоговых окнах теперь расположены в правом нижнем углу (ранее были в левом верхнем углу); изменить расположение кнопок нельзя.
- **Шаблоны экспорта:** улучшена выгрузка таблиц с одним столбцом в Word. <!-- 1720484 -->
- **Аутентификация:** доработаны значки указателя мыши на странице входа. <!-- 1442113 -->

### Инфраструктура {: #release_notes_5.0.20251231_infrastructure }

- **{{ apacheKafkaVariants }}:** добавлены параметры для настройки репликации, партиций и запросов. <!-- 123456 --> См.: _[Конфигурация экземпляра, компонентов ПО и служб. Настройка][configuration_files_linux]_
- **База данных:**
    - оптимизирована обработка объектов и наборов данных; <!-- git pull 14414 -->
    - улучшена конфигурация кэшей при запуске системы. <!-- git pull 14380 -->
- **Резервное копирование:**
    - усовершенствован запуск резервного копирования по расписанию, устранены известные проблемы; <!-- 10393891 -->
    - добавлен скрипт для выполнения резервного копирования в кластерной конфигурации.
- **Системные уведомления:** повышена производительность передачи сообщений. <!-- 10409857 -->
- **Кластер:** реализована остановка **{{ productName }}** при переходе кластера в неактивное состояние или режим только для чтения данных. <!-- git pull 14272 -->

### Мобильное приложение {: #release_notes_5.0.20251231_mobile }

- **Атрибуты и поля:**
    - улучшена обработка HTML-текста; <!-- 10446764 -->
    - улучшено отображение длинных значений в карточках: при переполнении теперь отображается троеточие; <!-- 10377329 -->
    - исправлено отображение меню действий в поле типа «Изображение» с одним прикреплённым файлом; <!-- 10382628 -->
    - реализованы отображение и выбор значений атрибутов типа «Цвет» (раньше эти поля были доступны только для чтения). <!-- 10376678, 10352555 -->
- **Кнопки:**
    - улучшена работа кнопок в таблицах на формах; <!-- 10378255 -->
    - оптимизировано отображение кнопок по условиям; <!-- 10399473 -->
    - устранена ошибка создания связанной записи по нажатии кнопки при наличии диалогового окна; <!-- 10408558 -->
    - унифицировано отображение кнопок в мобильном приложении и веб-интерфейса в формах, доступных только для чтения. <!-- 10364376 -->
- **Графики и диаграммы:** повышена стабильность отображения столбчатого графика на рабочем столе. <!-- 10434390 -->
- **Интерфейс:** исправлены стили на странице загрузки приложения. <!-- git pull 14475 -->

## Исправления {: #release_notes_5.0.20251231_fixes }

- **Регламенты и процессы:**
    - теперь экспортируются PNG-изображения в регламентах подпроцессов после преобразования процесса из архитектурного в автоматизированный; <!-- 10416508 -->
    - атрибуты моделей больше не хранят значения всех версий диаграмм; <!-- 10410887 -->
    - повышена стабильность длительно выполняющихся процессов. <!-- 10319599 -->
- **Формы:**
    - исправлено сохранение правил для форм и фильтров элементов в формах пользовательских задач, в которых используются системные атрибуты шаблона процесса; <!-- 10362774 -->
- **Обновление:**
    - реализовано прямое обновление ПО с версии 4.7.23499.0 на 5.0.24055.0; <!-- 10411697 -->
    - реализовано обновление версии ПО при наличии в базе данных аккаунтов с одинаковыми именами пользователей в разных регистрах (case-insensitive). <!-- 10380213 -->
- **Лицензирование:**
    - исправлен подсчёт статистики лицензий; <!-- 10383788 -->
    - улучшен контроль авторизации без лицензий. <!-- 10382214 -->
- **Производительность и стабильность:**
    - исправлена обработка исключений на таймерах; <!-- git pull 14270 -->
    - исправлены опечатки в журналах; <!-- git pull 14147 -->
    - ускорена проверка состояния кластера. <!-- git pull 14202 -->
