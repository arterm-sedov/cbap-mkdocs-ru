# Сведения о выпуске 5.0 от 31.12.2025 {: #release_notes_5.0.20251231 }

!!! danger "Внимание!"

    Если ваша версия **{{ productName }}** выпущена после 15.01.2026, её поведение может отличаться от версии, выпущенной 31.12.2025.

## Введение {: #release_notes_5.0.20251231_intro }

**{{ productName }}** версии 5.0 от 31.12.2025 включает ряд новых возможностей, улучшений и исправлений по сравнению с [версией 5.0 от 10.10.2025][release_notes_5.0.20251010].

## Ключевые изменения {: #release_notes_5.0.20251231_key_changes }

- **Настройка создания топиков {{ apacheKafkaVariants }}:** добавлены параметры конфигурации для управления созданием топиков очереди сообщений. <!-- 123456 -->

    См.: _[Конфигурация экземпляра, компонентов ПО и служб. Настройка][configuration_files_linux]_

- **Поддержка сервисов на Service Host (API gateway):** реализована поддержка частичного и полного переопределения асинхронных сервисов, поддержка Hub. <!-- 1915899 -->

- **Увеличение времени ожидания ответа адаптеров:** увеличено время ожидания ответа для HTTP-адаптера до 10 минут и для MySQL-адаптера до 5 минут. <!-- 1613345 -->

- **Механизм автоматического запуска адаптеров:** реализован механизм автоматического запуска адаптеров на старте AdapterHost, включая адаптеры, которые платформа уже отправляла. <!-- 1936565 -->

- **Обсуждения:** улучшена система обсуждений: добавлены системные уведомления о новых сообщениях, обновление счётчиков непрочитанных сообщений, перенос сервиса подписок в API gateway. <!-- 14376 -->

## Новые возможности и улучшения {: #release_notes_5.0.20251231_new_and_improved }

### Интерфейс {: #release_notes_5.0.20251231_ui }

- **Атрибут типа «Цвет»:** добавлен виджет выбора цвета для мобильного приложения. <!-- 10376678, 10352555 -->

- **Рабочий стол и дашборды:**
    - исправлено отображение высоты графиков на странице рабочего стола; <!-- 10261293 -->
    - удалён лишний скролл в настройке страницы боковой навигации; <!-- 10392976 -->
    - исправлено сохранение быстрых фильтров в дашбордах; <!-- 10411897 -->
    - улучшена валидация быстрых фильтров; <!-- 10365804 -->
    - исправлено отображение атрибутов в настройке детализации данных диаграммы; <!-- 10410978 -->
    - улучшена видимость цвета прошедших токенов в процессах.

- **Фильтры:** оптимизирована работа редактора выпадающих списков для уменьшения нагрузки на систему при работе с фильтрами ссылочных атрибутов. <!-- 10433067 -->

- **Навигация:** исправлено дублирование виджетов в колонках при редактировании страниц навигации. <!-- 10402713 -->

- **Таблицы и коллекции:** реализован перенос слов в заголовках атрибутов коллекций для улучшения отображения длинных названий. <!-- 10436143 -->

- **Шаблоны экспорта:** исправлена выгрузка таблиц с одним столбцом в Word. <!-- 1720484 -->

### Инфраструктура {: #release_notes_5.0.20251231_infrastructure }

- **Очередь сообщений:** добавлены параметры конфигурации для управления созданием топиков {{ apacheKafkaVariants }}:
    - `mq.replicationFactor` — параметр ReplicationFactor для создаваемого топика;
    - `mq.numPartitions` — параметр NumPartitions для создаваемого топика;
    - `mq.metadataTimeoutMsec` — таймаут для запроса метаданных. <!-- 123456 -->

- **База данных:**
    - реализован сбор объектов через Object API с использованием нативных запросов; <!-- 14414 -->
    - реализован набор данных на нативных запросах; <!-- 14096 -->
    - улучшена конфигурация кэшей: добавлена возможность автоматического применения новой конфигурации к существующим кэшам при запуске системы. <!-- 14380 -->

- **Резервное копирование:** добавлена валидация временных интервалов сессий резервного копирования для корректного выполнения по расписанию. <!-- 10393891 -->

- **Системные уведомления:** уменьшен размер сообщений по WebSocket путём удаления null-значений из сериализатора процессора запросов. <!-- 10409857 -->

- **Кластер:** реализована остановка платформы при переходе кластера в неактивное состояние или режим только чтения. <!-- 14272 -->

- **Сессии:** реализована поддержка идентификатора устройства в сессиях. <!-- 14119 -->

### Мобильное приложение {: #release_notes_5.0.20251231_mobile }

- **Атрибуты:**
    - исправлено сохранение значений HTML-атрибутов в мобильном приложении; <!-- 10446764 -->
    - исправлено отображение значений в карточках: значения теперь корректно уходят в троеточие при переполнении; <!-- 10377329 -->
    - исправлено отображение троеточия с функциями при наличии одной картинки в атрибуте типа «Изображение»; <!-- 10382628 -->
    - добавлен виджет выбора цвета для атрибута типа «Цвет». <!-- 10376678, 10352555 -->

- **Кнопки и операции:**
    - исправлена работа кнопок в коллекциях; <!-- 10378255 -->
    - исправлено отображение кнопок, удовлетворяющих условию отображения; <!-- 10399473 -->
    - исправлена ошибка создания связанной записи по кнопке при наличии диалогового окна; <!-- 10408558 -->
    - исправлено разное отображение кнопок в мобильном приложении и десктопе для форм только для чтения. <!-- 10364376 -->

- **Графики и диаграммы:** исправлено падение мобильного приложения при наличии столбчатого графика на рабочем столе. <!-- 10434390 -->

- **Интерфейс:** исправлены стили на странице загрузки. <!-- 14475 -->

- **Лицензирование:**

- **Производительность и стабильность:**

    - исправлены опечатки в логах; <!-- 14147 -->
    - исправлены скрипты отчётов; <!-- 14133 -->
