# Push-уведомления в браузере. Включение {: #browser_push_permissions }

## Введение {: #browser_push_permissions_intro }

Push‑уведомления позволяют **{{ productName }}** оперативно информировать вас о задачах и событиях, даже когда вкладка с сайтом **{{ productName }}** не активна.

По умолчанию большинство браузеров работает следующим образом:

- при первом обращении сайта браузер запрашивает ваше согласие на отправку уведомлений;
- если вы **разрешите** уведомления, сайт сможет отправлять уведомления, пока вы не измените решение;
- если вы **запретите** уведомления, сайт не сможет отправлять уведомления, пока вы вручную не разрешите их в настройках браузера;
- запросы на отправку уведомлений могут быть отключены по умолчанию и не отображаться при первом посещении сайта;
- некоторые браузеры могут автоматически подавлять навязчивые запросы на уведомления (например, в Chrome может действовать «тихий» режим запросов).

!!! note "Системные настройки уведомлений"

    - Windows/macOS: вывод уведомлений зависит от Центра уведомлений. Если в операционной системе для браузера отключены уведомления или включён режим «Не беспокоить», сообщения не отображаются, даже если они разрешены на сайте.
    - iOS: для сайтов нет отдельных разрешений. В настройках устройства можно только разрешить/запретить уведомления браузеру и веб‑сайтам в целом.

Здесь представлены краткие инструкции по включению push-уведомлений от **{{ productName }}** в популярных браузерах.

## Safari (macOS) {: #browser_push_permissions_safari }

1. Откройте «Safari» — «Настройки» — вкладка «Веб‑сайты» — раздел «Уведомления».
2. Включите флажок «Разрешить сайтам запрашивать разрешение на отправку push‑уведомлений».
3. В списке сайтов найдите адрес вашего сервера и выберите «Разрешить».
4. При необходимости вернитесь на сайт и обновите страницу.

Как отключить: в том же списке установите «Запретить» для сайта или снимите общий флажок запросов.

Источник: обсуждение на сайте Apple Community — [How to enable website notifications for Safari](https://discussions.apple.com/thread/255518428?sortBy=rank).

## Google Chrome (Windows/macOS/Linux) {: #browser_push_permissions_chrome }

### Способ 1. С помощью адресной строки

1. Перейдите на сайт с **{{ productName }}**.
2. Нажмите значок «замок» слева от адреса.
3. В блоке «Разрешения» задайте «Уведомления: Разрешить».

### Способ 2. С помощью настроек

1. В правом верхнем углу окна нажмите значок с тремя точками <i class="fa-light fa-ellipsis-vertical"></i>.
2. Выберите пункты «**Настройки**» <i class="fa-light fa-gear"></i> — «**Конфиденциальность и безопасность**» — «**Настройки сайтов**» — «**Уведомления**».
3. В разделе «**Разрешено показывать уведомления**» нажмите кнопку «**Добавить**».
4. Во всплывающем окне добавления сайта введите адрес сайта **{{ productName }}** и нажмите кнопку  «**Добавить**».

Справка Chrome: _[Как пользоваться уведомлениями](https://support.google.com/chrome/answer/3220216?hl=ru-RU&co=GENIE.Platform%3DDesktop)_

## Microsoft Edge (Windows/macOS) {: #browser_push_permissions_edge }

### Способ 1. С помощью адресной строки

1. Откройте сайт с **{{ productName }}**.
2. Нажмите значок «**Просмотр сведений о сайте**» <i class="fa-light fa-lock-keyhole"></i> слева от адреса сайта.
3. В раскрывающемся меню выберите пункт «**Разрешения для этого сайта**».
4. В меню «**Уведомления**» выберите пункт «**Разрешить**».

### Способ 2. С помощью настроек

1. Выберите меню «**Параметры и другое**» <i class="fa-light fa-ellipsis"></i> в правом верхнем углу окна браузера.
2. Выберите пункты «**Параметры**» <i class="fa-light fa-gear"></i> — «**Конфиденциальность, поиск и службы**» — «**Разрешения сайтов**» — «**Все разрешения**» — «**Уведомления**» — «**Особое поведение**».
3. В разделе «**Разрешить**» нажмите кнопку «**Добавить сайт**»
4. Во всплывающем окне добавления сайта введите адрес сайта **{{ productName }}** и нажмите кнопку  «**Добавить**».

Справка Edge: _[Управление уведомлениями сайтов в Microsoft Edge](https://support.microsoft.com/ru-ru/microsoft-edge/%D1%83%D0%BF%D1%80%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5-%D1%83%D0%B2%D0%B5%D0%B4%D0%BE%D0%BC%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F%D0%BC%D0%B8-%D1%81%D0%B0%D0%B9%D1%82%D0%BE%D0%B2-%D0%B2-microsoft-edge-0c555609-5bf2-479d-a59d-fb30a0b80b2b#:~:text=Manage%20notifications%20from%20your%20browser,this%20menu%20and%20choosing%20Allow.)_

## Mozilla Firefox (Windows/macOS/Linux) {: #browser_push_permissions_firefox }

### Способ 1. Через информацию о сайте

1. Перейдите на сайт с **{{ productName }}**.
2. Нажмите на значок «замок» — стрелка — «Дополнительная информация».
3. На вкладке «Разрешения» включите «Отправлять уведомления: Разрешить».

### Способ 2. С помощью настроек

1. «Настройки» — «Приватность и защита».
2. В разделе «Разрешения» нажмите «Параметры…» рядом с «Уведомления» и добавьте сайт в «Разрешить».

Справка: [Web Push notifications in Firefox](https://support.mozilla.org/en-US/kb/push-notifications-firefox).

## Яндекс Браузер (Windows/macOS/Android) {: #browser_push_permissions_yandex }

1. При первом посещении сайта вы увидите запрос на показ уведомлений — выберите «Разрешить».
2. Чтобы изменить решение позже: «Настройки» — «Сайты» — «Уведомления» и добавьте адрес сервера в «Разрешить» или «Блокировать».
3. На Android системные настройки имеют приоритет: если уведомления отключены в ОС, они не будут приходить даже при разрешении в браузере. На iOS настройка уведомлений для отдельных сайтов недоступна — можно только полностью запретить уведомления веб‑сайтам в настройках устройства.

Справка: [Показ уведомлений от сайтов и приложений (Яндекс Поддержка)](https://yandex.ru/support/common/ru/browsers-settings/notifications).

## Проверка работы уведомлений в {{ productName }} {: #browser_push_permissions_testing }

1. Разрешите уведомления для адреса вашего сервера **{{ productName }}** в браузере по инструкциям выше.
2. В веб‑версии выполните действие, которое отправляет уведомление (например, получение новой задачи) или дождитесь push‑сообщения.
3. Убедитесь, что в системном Центре уведомлений вашего устройства уведомления для соответствующего браузера не заблокированы и режим «Не беспокоить» отключён.

## Частые проблемы и решения {: #browser_push_permissions_troubleshooting }

- Вы ранее нажали «Запретить». Решение: откройте настройки сайта и измените «Уведомления: Разрешить», затем перезагрузите страницу.
- Браузер/ОС всё равно не показывает уведомления. Решение: проверьте системные настройки уведомлений для браузера и отключите режим «Не беспокоить».
- Safari не показывает запрос. Решение: включите общий флажок «Разрешить сайтам запрашивать…» в Safari и только затем обновите сайт.
