# Мобильное приложение. Использование {: #mobile_app_use }

## Введение {: #mobile_app_use_intro }

Для **{{ productName }}** доступно мобильное приложение **{{ productNameMobile }}**.

**{{ productNameMobile }}** предназначено для использования готовых настроенных приложений на основе **{{ productName }}**.

С помощью **{{ productNameMobile }}** можно:

- заполнять формы;
- работать с записями;
- выполнять задачи;
- просматривать таблицы;
- искать данные;
- сканировать QR-коды и NFC/RFID-метки;
- прикреплять файлы;
- просматривать рабочий стол и информационные страницы (дашборды);
- получать push-уведомления о задачах и обсуждениях.

Все изменения данных автоматически синхронизируются между **{{ productNameMobile }}** и веб-версией **{{ productName }}**.

!!! tip "Совет"

    Для оптимального использования **{{ productNameMobile }}** рекомендуется:
    
    - регулярно проверять наличие обновлений **{{ productNameMobile }}**;
    - разрешить push-уведомления в настройках мобильного устройства;
    - использовать стабильное подключение к интернету для надёжной синхронизации данных с веб-версией **{{ productName }}**.

!!! warning "Внимание!"

    **{{ productNameMobile }}** не позволяет настраивать административные параметры **{{ productName }}**, а также редактировать приложения, шаблоны, атрибуты, формы, таблицы и т. п.

    Такую настройку следует осуществлять с помощью веб-версии **{{ productName }}**.

## Первый вход в {{ productNameMobile }} {: #mobile_app_use_login }

1. Запустите **{{ productNameMobile }}**.
2. На странице входа укажите **адрес сервера {{ productName }}**, например:

    ```
    https://<your-host>
    ```

3. Укажите **эл. почту или имя пользователя**.
4. Укажите **пароль**.
5. Нажмите кнопку «**Войти**».
6. Если для вашего аккаунта включена двухфакторная аутентификация, то после входа вам будет отправлено письмо с кодом подтверждения. Введите полученный код из четырёх цифр в течение 5 минут после его отправки.

    !!! warning "Логика двухфакторной аутентификации"

        Если для аккаунта включена двухфакторная аутентификация, код подтверждения потребуется вводить при каждом входе перед вводом кода доступа к **{{ productNameMobile }}**.

         См. _«[Двухфакторная аутентификация. Настройка][two_factor_authentication]»_.

7. После входа и прохождения двухфакторной аутентификации задайте подтвердите код доступа из четырёх цифр. Этот код доступа будет использоваться для последующего входа в **{{ productNameMobile }}** вместо ввода имени пользователя и пароля.
8. Отобразится **[Главная](#mobile_app_use_home)** страница.

!!! warning "Внимание!"

    **{{ productNameMobile }}** не позволяет создавать новые аккаунты и выполнять самостоятельную регистрацию.

    Можно использовать только имеющиеся аккаунты, настроенные в веб-версии **{{ productName }}**.

## Добавление авторизованного аккаунта {: #mobile_app_use_add_account }

1. Запустите **{{ productNameMobile }}**.
2. Выполните [вход](#mobile_app_use_login) в имеющийся аккаунт.
3. Выберите вкладку «**Меню**».
4. Нажмите кнопку «**Выйти**».
5. Отобразится страница выбора аккаунта.
6. Нажмите кнопку «**Добавить аккаунт**».
7. Выполните [вход](#mobile_app_use_login) в новый аккаунт.

## Удаление авторизации аккаунта {: #mobile_app_use_remove_account }

1. Запустите **{{ productNameMobile }}**.
2. Выполните [вход](#mobile_app_use_login) в имеющийся аккаунт.
3. Выберите вкладку «**Меню**».
4. Нажмите кнопку «**Выйти**».
5. Отобразится страница выбора аккаунта.
6. Проведите влево по строке аккаунта, авторизацию которого требуется удалить.
7. Нажмите кнопку «**Удалить**».

## Просмотр Рабочего стола <i class="fa-light fa-house"></i> {: #mobile_app_use_home }

1. Нажмите кнопку «**Главная**» <i class="fa-light fa-house"></i>.
2. Отобразится **Рабочий стол**. См. _«[Рабочий стол. Использование][desktop]»_.

Настроить **Рабочий стол** можно с помощью веб-версии **{{ productName }}**. См. _«[Рабочий стол. Настройка][desktop_setup]»_.

## Навигация <i class="fa-light fa-bars"></i> {: #mobile_app_use_navigation }

1. Нажмите кнопку <i class="fa-light fa-bars"></i> в верхнем левом углу.
2. Отобразится панель навигации.
3. Выберите требуемую страницу.
4. Чтобы перейти к предыдущей странице, нажмите значок <i class="fa-light fa-angle-left"></i> в верхнем левом углу.
5. Если отображаются карточки или таблица, нажмите значок <i class="fa-light fa-ellipsis"></i> в верхнем правом углу, чтобы настроить представление. См. _«[Просмотр карточек и таблиц](#mobile_app_use_cards)»_.

Настроить панель навигации можно в веб-версии **{{ productName }}**. См. _«[Панель навигации. Настройка][navigation_sections_setup]»_.

## Поиск данных <i class="fa-light fa-magnifying-glass"></i> {: #mobile_app_use_search }

**{{ productNameMobile }}** позволяет искать записи по значениям полей (атрибутов) и именам файлов, прикреплённых к записям.

1. Нажмите кнопку «**Поиск**» <i class="fa-light fa-magnifying-glass"></i>.
2. Введите в поле поиска искомый текст.
3. Отобразится список всех записей, где было найдено искомое слово.
4. Выберите вкладку «**Поля**».
5. Отобразится список записей, в которых были найдены искомые слова.
6. Выберите вкладку «**Файлы**».
7. Отобразится список записей с прикреплёнными файлами, имена которых содержат искомые слова.

## Сканирование QR-кодов <i class="fa-light fa-qrcode"></i> {: #mobile_app_use_qr_code }

!!! warning "Доступ к камере"

    Для сканирования QR-кодов необходимо разрешить **{{ productNameMobile }}** доступ к камере в настройках мобильного устройства.

!!! warning "Логика сканирования QR-кодов и NFC/RFID-меток"

    С помощью функции «**QR-код**» можно считывать гиперссылки на страницы, задачи, формы и таблицы, имеющиеся на сервере **{{ productName }}**, на который выполнен вход.
    
    Чтобы считать гиперссылку на внешний ресурс (например, веб-страницу), эта ссылка должна присутствовать в панели навигации для текущего пользователя. См. _«[Панель навигации. Настройка][navigation_sections_setup]»_.

1. Нажмите кнопку «**QR-код**» <i class="fa-light fa-qrcode"></i>.
2. Разрешите использование камеры для **{{ productNameMobile }}**.
3. Наведите камеру на QR-код.
4. Откроется страница, на которую ссылается QR-код.

## Сканирование NFC/RFID-меток <i class="fa-light fa-qrcode"></i> {: #mobile_app_use_nfc }

При использовании **{{ productNameMobile }}** на специализированных устройствах можно считывать считывать NFC/RFID-метки, содержащие гиперссылки, аналогично сканированию QR-кодов.

1. Нажмите кнопку «**QR-код**» <i class="fa-light fa-qrcode"></i>.
2. Разрешите доступ к NFC-сканеру для **{{ productNameMobile }}**.
3. Поднесите устройство к NFC/RFID-метке.
4. Откроется страница, на которую ссылается метка.

## Просмотр уведомлений <i class="fa-light fa-bell"></i> {: #mobile_app_use_notifications }

1. Нажмите кнопку «**Уведомления**» <i class="fa-light fa-bell"></i>.
2. Откроется вкладка «**События**», и отобразится список уведомлений. Непросмотренные события будут выделены цветом.
3. Введите значение в поле «**Поиск**» для фильтрации событий.
4. Перейдите на вкладку «**Обсуждения**».
5. Откроется список обсуждений, в которых участвует пользователь. Непросмотренные обсуждения будут выделены цветом.
6. Введите значение в поле «**Поиск**» для фильтрации обсуждений.
7. Чтобы удалить все уведомления, нажмите кнопку «**Очистить**» в верхнем правом углу.

!!! note "Push-уведомления"

    **{{ productNameMobile }}** может выводить push-уведомления, которые позволяют оперативно узнавать о новых задачах и сообщениях, даже когда приложение не открыто.
    
    Для получения push-уведомлений необходимо разрешить их при установке **{{ productNameMobile }}** и в настройках мобильного устройства.

## Использование меню <i class="fa-light fa-gear"></i>  {: #mobile_app_use_menu }

1. Нажмите кнопку «**Меню**» <i class="fa-light fa-gear"></i>.
2. Отобразится краткая информация об аккаунте, в который выполнен вход, а также имя и версия сервера **{{ productName }}**.
3. Чтобы сменить код доступа к **{{ productNameMobile }}**, выберите пункт «**Сменить код доступа**».
4. Чтобы просмотреть подробные сведения об аккаунте, выберите пункт «**Аккаунт**».
5. Чтобы перейти к базе знаний **{{ companyName }}**, выберите пункт «**Справка**».
6. Чтобы выйти из аккаунта и перейти к странице выбор аккаунта, нажмите пункт «**Выйти**».

## Просмотр карточек и таблиц {: #mobile_app_use_cards }

Списки записей в **{{ productNameMobile }}** могут отображаться в виде в виде карточек (по умолчанию) и таблиц. См. _«[Карточки. Использование][cards_view]»_ и _«[Таблицы. Использование][table_personal_use]»_.

1. Откройте требуемый список (таблицу).
2. Нажмите троеточие <i class="fa-light fa-ellipsis"></i> в верхнем правом углу.
3. Выберите требуемое представление записей: «**Таблица**» <i class="fa-light fa-table"></i> или «**Карточки**» <i class="fa-light fa-border-all"></i>.

### Сортировка

1. Нажмите троеточие <i class="fa-light fa-ellipsis"></i> в верхнем правом углу.
2. Выберите пункт «**Сортировка**» <i class="fa-light fa-arrow-down-a-z"></i>.
3. Отобразится всплывающее меню сортировки.
4. Выберите поле карточки (столбец таблицы) для сортировки записей по нему.
5. Закройте всплывающее меню.
6. Список записей будет отсортирован заданным образом.

### Настройка внешнего вида

1. Нажмите троеточие <i class="fa-light fa-ellipsis"></i>.
2. Выберите пункт «**Внешний вид**» <i class="fa-light fa-gear"></i>.
3. Отобразится страница настройки внешнего вида.
4. Выберите поля карточки (или столбцы таблицы), которые должны отображаться.
5. Выберите количество карточек или строк, которое должно отображаться на одной странице.
6. Нажмите значок <i class="fa-light fa-angle-left"></i> в верхнем левом углу.
7. Список записей будет отформатирован заданным образом.

### Фильтрация и группировка

1. Нажмите троеточие <i class="fa-light fa-ellipsis"></i>.
2. Выберите пункт «**Фильтры и группировка**» <i class="fa-light fa-filter"></i>.
3. Отобразится страница настройки фильтрации записей.
4. Выберите поле карточки (столбец таблицы).
5. Настройте фильтрацию и группировку для этого поля (столбца)
6. Нажмите значок <i class="fa-light fa-angle-left"></i> в верхнем левом углу.
7. Список записей будет отфильтрован заданным образом.

## Просмотр записей

1. Откройте запись или задачу, выбрав её карточку или строку в таблице.
2. Отобразится форма записи.
3. Чтобы перейти к определённой области записи, нажмите её название вверху страницы.
4. Заполните поля формы, если требуется.
5. Нажмите кнопку «**Сохранить**».
6. При необходимости перейдите к [обсуждению записи](#mobile_app_use_discussion).

## Обсуждение записей {: #mobile_app_use_discussion }

См. также _[подробные инструкции по использованию обсуждений][discussion_use]_.

1. Откройте запись (форму или задачу).
2. Нажмите значок <i class="fa-light fa-message-dots"></i> в верхнем правом углу.
3. Отобразится обсуждение для данной записи.
4. Введите текст сообщения под обсуждением.
5. Чтобы прикрепить файл, нажмите значок <i class="fa-light fa-paperclip"></i>.
6. Выберите файл из галереи или сделайте фото с помощью камеры.
7. Нажмите кнопку «**Отправить**».
