# Урок 2. Введение, рабочий стол, роли, навигация, дашборды {: #tutorial_architect_lesson_2 }

## Введение {: #tutorial_architect_lesson_2_intro }

В этом уроке вы научитесь быстро ориентироваться в интерфейсе платформы, использовать рабочий стол, быстрые действия и аналитику для решения ежедневных задач. 

Вы освоите:

- Оценку состояния архитектуры через дашборды и отчёты.
- Использование быстрых действий для запуска ключевых операций.
- Настройку ролей и прав доступа для эффективного разграничения обязанностей.

Эти навыки помогут повысить прозрачность процессов и быстро реагировать на изменения.

**Предусловия:** пройден урок 1 «Обзор возможностей платформы и понятий BPMS».

**Расчётная продолжительность:** 45 мин.

!!! warning "Бизнес-логика"

    Быстрая ориентация в интерфейсе платформы и понимание ролей пользователей — ключ к эффективному старту работы с корпоративной архитектурой. Освоив рабочий стол, быстрые действия и аналитику, вы сможете минимизировать ошибки, быстро реагировать на изменения и уверенно выполнять задачи как архитектор или сотрудник.

## Определения {: #tutorial_architect_lesson_2_definitions }

- **Архитектор** — пользователь, моделирующий процессы, настраивающий оргструктуру, согласующий и публикующий изменения.
- **Сотрудник** — пользователь, просматривающий оргструктуру, участвующий в согласованиях и аудитах, работающий с задачами и документами.
- **Рабочий стол** — стартовая страница с быстрыми действиями и аналитикой.
- **Быстрые действия** — плитки для быстрого запуска ключевых операций.
- **Дашборды** — визуализация процессов, заявок, аудитов, динамики изменений.
- **Навигация** — переход между основными разделами платформы.

## 4. Быстрые действия и виджеты {: #tutorial_architect_lesson_2_quick_actions }

!!! note "Практика"

    Откройте форму создания согласования через быстрые действия. Кликните по диаграмме в отчёте, чтобы перейти к деталям.

_![Плитки быстрых действий](img/lesson_2_quick_actions.png)_

Быстрые действия — это плитки на рабочем столе, которые позволяют быстро запускать ключевые операции. Например, создание согласования или аудита занимает всего пару кликов. Виджеты отображают актуальную аналитику: количество процессов, заявок, аудитов, динамику изменений.

1. Откройте форму создания согласования через быстрые действия.
2. Кликните по диаграмме в отчёте, чтобы перейти к деталям.

## Дашборды и аналитика {: #tutorial_architect_lesson_2_dashboards }

_![Пример дашборда](img/lesson_2_dashboard_example.png)_

Дашборды и отчёты — мощный инструмент для анализа процессов и оргструктуры. Вы можете видеть динамику заявок, статусы аудитов, использовать фильтры по периоду, юрлицу или ИТ-системе. Клик по диаграмме позволяет перейти к подробной информации.
