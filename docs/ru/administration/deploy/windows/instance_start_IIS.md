# Запуск экземпляра системы с помощью Диспетчера служб IIS {: #instance_start_IIS }

1. Откройте Диспетчер служб IIS.
2. Выберите сайт в левом дереве.
3. Нажмите кнопку «**Запустить**» в правой панели.
