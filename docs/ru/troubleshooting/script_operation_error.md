# Ошибка выполнения скриптовой операции {: #script_operation_error }

## Описание проблемы {: #script_operation_error_intro }

При настройке кнопки или сценария с C#-скриптом важно составить безупречный скрипт.

Неправильно прописанная логика, неверные системные имена шаблонов или атрибутов и т.&nbsp;п. могут привести к ошибке выполнения скрипта.

## Решение {: #solution }

Если в C#-скрипте возникла ошибка, обратите внимание на следующие факторы:

1. Проверьте правильность написания скрипта.
2. Проверьте разрешение в ролях пользователей. Важно, чтобы у ролей были права как на саму кнопку, так и на все шаблоны, прописанные в скрипте.
