# Неверная ссылка на инстанс {: #invalid_instance_reference }

## Описание проблемы {: #invalid_instance_reference_intro }

Если пользователю на почту пришло системное уведомление о назначенной на него задаче, но при переходе по ссылке на задачу система либо выдаёт ошибку, либо открывается некорректный адрес, то неправильно настроены системные уведомления или глобальная конфигурация.

## Решение {: #solution }

1. Проверьте на адрес сервера, который указан в глобальной конфигурации. Он должен совпадать с URL, на котором вы работаете с **{{ productName }}**.
2. Проверьте настройки уведомлений: общих, системных и процессных.
