# Невозможно вынести из ссылки на форму ряд элементов {: #extract_form_elements }

## Описание проблемы {: #extract_form_elements_intro }

В **{{ productName }}** предусмотрены ссылочные атрибуты типа «**Запись**».

В свойствах атрибута можно установить флажок «**Хранить несколько значений**», который задаёт сколько ID записей атрибут может хранить: одну или несколько.

Этот параметр определяет, на сколько записей в **связанном шаблоне** может ссылаться запись в текущем шаблоне.

Кроме того, этот параметр влияет на возможность добавления элементов из **связанного шаблона** на формы и в таблицы текущего шаблона.

## Решение {: #solution }

- Если атрибут типа «**Запись**» может хранить только одно значение, на форму шаблона можно поместить любой элемент **связанного шаблона** записи. При этом одной записи в текущем шаблоне записи соответствует одна запись в связанном шаблоне.
- Если атрибут типа «**Запись**» может **хранить несколько значений**, то не все элементы **связанного шаблона** можно перенести на форму текущего шаблона.
    - Так, например, на форму текущего шаблона не получится поместить форму из связанного шаблона. Это связано с тем, что атрибут хранит ID нескольких записей, а форма относится к одному шаблону.
    - Атрибуты связанного шаблона также нельзя поместить как поля на форме текущего шаблона. Их можно поместить на форму только в качестве **столбцов таблицы**, изменив **представление** атрибута типа «**Запись**» на **таблицу**.
