## Руководства к рекомендованной текущей версии ПО {: #portal_index_current_guides }

В следующих разделах представлены инструкции по использованию ПО **{{ productName }}** текущей рекомендованной версии **{{ productVersion }}**.

- [**{{ productName }}**. Версия {{ productVersion }} (текущая рекомендованная)][platform_v6_cat]
    - [Общая информация][general_cat] — описание ПО и основ его использования, часто задаваемые вопросы и т. д.
    - [Обучение][tutorials_cat] — уроки для быстрого освоения базовых возможностей ПО.
    - [Руководство пользователя][user_guide_cat] — полное руководство по использованию ПО.
        - [Интеграции][integrations_cat] — руководство по интеграции ПО с внешними системами.
        - [Моделирование бизнес-архитектуры][architect_cat] — руководство по использованию модуля «**{{ productNameArchitect }}**».
    - [Администрирование][product_administration_cat] — руководство по использованию встроенных административных функций ПО.
    - [Руководство системного администратора][admin_guide_cat] — руководство по развёртыванию, резервному копированию, восстановлению и поддержанию работоспособности ПО.
    - [Руководство разработчика][developer_guide_toc] — описание возможностей программного обращения к функциям ПО.
        - [Руководство по скриптам на C#][csharp_developer_guide_cat]
        - [Руководство по API][api_developer_guide_cat]
        - [Руководство по языку N3][n3_developer_guide_cat]
    - [Помощь в настройке][examples_cat] — статьи, описывающие настройку какого-либо решения для ПО.
    - [Выражения для вычислений][expressioons_cat] — примеры формул, выражений на N3 и скриптов на C#, которые можно использовать внутри ПО.
    - [Решение проблем][troubleshooting_cat] — статьи, отвечающие на наиболее распространенные вопросы пользователей.
