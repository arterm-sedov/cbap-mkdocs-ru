# Атрибут типа «Текст». Настройка пользовательской маски {: #text_attribute_mask }

Маска ввода представляет собой строку, позволяющую ввести значение допустимого формата символов. В **{{ productName }}** есть как заранее настроенные маски (ИНН, адрес эл.&nbsp;почты и т.&nbsp;д.), так и вариант «**Пользовательская маска**», который используется в случае необходимости настройки атрибута с вводом данных в определённом формате.

## Прикладная задача {: #text_attribute_mask_use_case }

Требуется ограничить в текстовом атрибуте _«Наименование»_ ввод только латинских букв. При этом длина строки не может превышать 10 символов.

## Порядок настройки {: #text_attribute_mask_setup }

1. Откройте атрибут _«Наименование»_.
2. Выберите **формат отображения** «**Пользовательская маска**».
3. В поле «**Регулярное выражение маски**» вставьте следующее:

    ```text
    [A-Za-z]{10}
    ```

4. Нажмите кнопку «**Сохранить**».

При заполнении данного поля пользователи смогут вводить только латинские буквы в верхнем и нижнем регистре — не более 10 букв.

## Синтаксис {: #text_attribute_mask_sintaksis }

Формат пользовательской маски задаётся регулярным выражением и может содержать ограничение на буквы, цифры и символы, а также разделители и фиксированные символы (как, например, у маски телефонного номера). Фиксированные символы отделяются знаком `\` до и после символа.

| Элемент | Описание |
| --- | --- |
| `[A-Za-z]` | Латинские буквы в верхнем и нижнем регистре |
| `{10}` | Ровно 10 символов указанного типа |
