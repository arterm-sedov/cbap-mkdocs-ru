# Импорт связанных данных из таблицы Excel {: #import_excel_data }

Здесь приведён пример импорта связанных данных из файла Excel в **{{ productName }}**.

Такой импорт удобно использовать, когда требуется загрузить записи в связанный шаблон и сразу отобразить их в таблице на форме основной записи.

## Принцип работы {: #import_excel_data_logic }

Таблица на форме записи отображает записи из связанного шаблона. Каждая строка такой таблицы является отдельной записью, которая ссылается на основную запись.

Чтобы заполнить таблицу на форме с помощью Excel:

1. Импортируйте файл Excel в связанный шаблон записи.
2. Укажите в отдельном столбце файла значение для связи с основной записью:

    - **ID** основной записи;
    - значение её [атрибута-заголовка][attribute_displayed], если оно используется как формат сопоставления **Name**.

После импорта записи будут связаны с основной записью, а таблица на её форме заполнится автоматически.

## Пример настройки {: #import_excel_data_example }

В примере требуется импортировать заказы клиентов из Excel.

В приложении созданы два шаблона записи:

- _Клиенты_ — основной шаблон записи;
- _Заказы_ — связанный шаблон записи.

В шаблоне _«Заказы»_ настроен атрибут типа «**Запись**», который ссылается на шаблон _«Клиенты»_. В шаблоне _«Клиенты»_ настроен связанный атрибут _«Заказы клиента»_ с включённым хранением нескольких значений.

### Подготовка файла Excel {: #import_excel_data_prepare_file }

Подготовьте файл Excel со столбцами заказов и отдельным столбцом для связи с клиентом.

_![Пример таблицы Excel для импорта заказов](https://kb.comindware.ru/assets/excel_table_import.png)_

В столбце _«Клиент»_ укажите названия клиентов, уже созданных в **{{ productName }}**. Эти значения должны совпадать со значениями атрибута-заголовка записей в шаблоне _«Клиенты»_.

### Импорт заказов {: #import_excel_data_import_orders }

1. Откройте шаблон записи _«Заказы»_.
2. Перейдите на вкладку «**Импорт**».
3. Загрузите подготовленный файл Excel.
4. Настройте сопоставление столбцов Excel с атрибутами шаблона _«Заказы»_.
5. Для столбца _«Клиент»_ выберите атрибут типа «**Запись**», который ссылается на шаблон _«Клиенты»_.
6. В поле «**Формат**» для столбца _«Клиент»_ выберите **Name**.
7. Для столбца _«Сумма»_ выберите формат «**Валюта**».
8. Нажмите кнопку «**Загрузить**».

_![Настройка импорта таблицы Excel](https://kb.comindware.ru/assets/2021-12-30_13h46_19.png)_

## Результат {: #import_excel_data_result }

После импорта в шаблоне _«Заказы»_ появятся новые записи со связью с соответствующими клиентами.

_![Импортированные записи в шаблоне «Заказы»](https://kb.comindware.ru/assets/2021-12-30_13h48_11.png)_

Таблица _«Заказы клиента»_ на форме записи в шаблоне _«Клиенты»_ заполнится автоматически.

_![Таблица заказов клиента](https://kb.comindware.ru/assets/2021-12-30_13h49_38.png)_
