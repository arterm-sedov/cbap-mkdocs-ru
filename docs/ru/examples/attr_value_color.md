# Атрибут типа «Текст». Настройка цвета отображения значения {: #attr_value_color }

Все атрибуты в системе отображаются в определённом формате. Настроить определённый шрифт или размер шрифта для значений конкретного атрибута нельзя. Если же вы хотите, чтобы какой-либо атрибут в таблице окрашивался определённым цветом по заданному условию (например, если задача просрочена, выделить значение красным цветом, в противном случае — зелёным), то это возможно через атрибут типа «**Текст**» с форматом отображения «**HTML-текст**».

## Порядок настройки {: #attr_value_color_setup }

1. Создайте атрибут типа «**Текст**», выберите формат его отображения «**HTML-текст**». Отметьте флаг «**Вычисляемый**».

2. В поле «**Вычисляемое выражение**» выберите «**Формула**» и вставьте выражение:

```sql
FORMAT(
IF(EQUALS($StatusRef->name,"Выполнено"),
"<p><span style='color:#000000'><span style='background-color:#3ee028'>{0}</span></span></p>",
"<p><span style='color:#000000'><span style='background-color:#f02626'>{0}</span></span></p>"),
LIST($StatusRef->name))
```

## Синтаксис {: #attr_value_color_sintaksis }

Для решения задачи используется функция `FORMAT()`, которая форматирует значение из нужного атрибута (в данном примере, _«Статус»_). Далее прописывается условие с помощью функции `IF()`. В нашем случае это _«Статус»_ равен _«Выполнено»_. После этого, если условие выполняется, то задний фон окрашивается в зелёный цвет, а текст — в чёрный. Если условие не выполняется, то задний фон окрашивается в красный цвет, а текст — в чёрный.

Для окрашивания заднего фона и текста используется HTML — язык гипертекстовой разметки. Цвет заднего фона можно поменять в значении `background-color:#f02626`. Цвет текста можно поменять в значении `color:#000000`.

В конце выражения в `LIST()` прописывается атрибут, значение которого нужно окрасить. В нашем случае это `$StatusRef->name`. Нужно заметить, что окрасится не сам атрибут _«Статус»_, а отдельный вычисляемый атрибут, который вы создали.

!!! note "Примечание"

    Ограничений в настройке нет: можно окрашивать только текст, только фон, добавлять дополнительные HTML-значки и т.&nbsp;д.
