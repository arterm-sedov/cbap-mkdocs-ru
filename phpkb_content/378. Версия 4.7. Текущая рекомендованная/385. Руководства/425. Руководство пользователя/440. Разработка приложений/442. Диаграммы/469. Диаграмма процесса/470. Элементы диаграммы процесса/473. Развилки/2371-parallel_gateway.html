Определения

Развилка «и» используется двумя способами:

  • Расходящаяся развилка «и» — у развилки имеется несколько исходящих потоков. При входе в расходящуюся развилку «и» токена из неё выходит одновременно несколько токенов по всем исходящим потокам.
  • Сходящаяся развилка «и» — у развилки имеется один исходящий поток. Сходящаяся развилка «и» ожидает входа токенов по всем входящим потокам, после чего из развилки выходит один токен по исходящему потоку.

Примечание

Количество входящих и исходящих потоков у развилки «и» может быть любым. Но не рекомендуется одновременно использовать несколько входящих и исходящих потоков. Рекомендуется соединять развилку либо с одним входящим потоком, либо с одним исходящим.
