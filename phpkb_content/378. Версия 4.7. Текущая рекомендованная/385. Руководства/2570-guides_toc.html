Руководства. Содержание раздела

  • Руководство пользователя — полное руководство по использованию ПО.
    • Интеграции — руководство по интеграции ПО с внешними системами.
    • Моделирование бизнес-архитектуры — руководство по использованию модуля Comindware Architect.
  • Помощь в настройке — примеры настройки приложений.
  • Администрирование — руководство по использованию встроенных административных функций ПО.
  • Руководство системного администратора — руководство развёртыванию, резервному копированию, восстановлению и поддержанию работоспособности ПО.
  • Руководство разработчика — описание возможностей программного обращения к функциям ПО.
    • Руководство по скриптам на C#
    • Руководство по API
    • Руководство по языку N3
