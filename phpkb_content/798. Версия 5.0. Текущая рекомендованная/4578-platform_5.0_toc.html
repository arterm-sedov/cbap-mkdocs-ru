Comindware Platform. Версия 5.0. Содержание раздела

Содержание раздела

  • Общая информация — описание ПО и основ его использования, часто задаваемые вопросы и т. д.
  • Обучение — уроки для быстрого освоения базовых возможностей ПО.
  • Руководство пользователя — полное руководство по использованию ПО.
    • Интеграции — руководство по интеграции ПО с внешними системами.
    • Моделирование бизнес-архитектуры — руководство по использованию модуля «Корпоративная архитектура».
  • Администрирование — руководство по использованию встроенных административных функций ПО.
  • Руководство системного администратора — руководство развёртыванию, резервному копированию, восстановлению и поддержанию работоспособности ПО.
  • Руководство разработчика — описание возможностей программного обращения к функциям ПО.
    • Руководство по скриптам на C#
    • Руководство по API
    • Руководство по языку N3
  • Помощь в настройке — статьи, описывающие настройку какого-либо решения для ПО.
  • Выражения для вычислений — примеры формул, выражений на N3 и скриптов на C#, которые можно использовать внутри ПО.
  • Решение проблем — статьи, отвечающие на наиболее распространенные вопросы пользователей.
