# {{ productName }} 5.0. Перечень стороннего программного обеспечения для Linux

Введение

Некоторые функции **{{ productName }}** (далее «**Система**») реализованы с использованием сторонних библиотек.

Для обеспечения работы Системы используются сторонние сервисы, такие как веб-сервер, СУБД, обратный прокси-сервер и т. п.

В настоящей статье представлен перечень стороннего программного обеспечения (ПО), которое поставляется вместе с **{{ productName }}** версии 5.0 для операционных систем (ОС) семейства Linux: «Альт Сервер» 10.0–10.1, Astra Linux SE 1.7.5 «Смоленск».

Состав стороннего ПО для других операционных систем может отличаться.
