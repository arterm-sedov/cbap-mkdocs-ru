# Невозможно просмотреть историю значений вычисляемого атрибута

## Описание проблемы

В свойствах атрибута есть флажок «**Записывать изменения в историю**». Если он установлен, любые изменения в атрибуте будут регистрироваться.

Этот флажок можно установить только для невычисляемых атрибутов.

По вычисляемым атрибутам история не сохраняется, так как их значения вычисляются на летку и является непостоянным.

Все изменения значений атрибутов сохраняются в журнале изменений записи.

Для просмотра журнала изменений перейдите в конкретную запись, нажмите кнопку «**Информация**» *‌* сверху справа и выберите пункт «**Журнал изменений**».

## Решение

Если всё же требуется хранить историю значений вычисляемого атрибута:

1. Создайте дополнительный служебный невычисляемый атрибут.
2. Вручную (или с помощью сценария) записывайте в служебный атрибут значения вычисляемого.
3. В журнале изменений будет видна история значений служебного атрибута, совпадающая с изменениями вычисляемого.
